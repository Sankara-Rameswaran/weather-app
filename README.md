# Weather App

A clean and responsive weather application built with React.js that provides real-time weather information for any city. The app fetches live weather data using the OpenWeather API and displays key details such as temperature, weather conditions, humidity, and wind speed.

## Features

- Search weather by city name
- Real-time weather data
- Current temperature display
- Weather condition information
- Humidity and wind speed details
- Responsive and user-friendly interface

## Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- OpenWeather API

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm start
```

The application will run on `http://localhost:3000`.

## API Used

This project uses the **OpenWeather API** to fetch real-time weather data.

Get your API key from: https://openweathermap.org/api
